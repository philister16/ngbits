Flash

This bit displays flash messages (also called snackbars or toasts) from only a service. There is no need to place a component in your views.

You need to import BrowserAnimationsModule in your root module!

Examples

For convenience the service also provides contextual methods which by default are styled with bootstrap's alert classes. The css classes can be overriden just like with custom flash messages, however.

API

NgbitFlashModule Module
NgbitFlashService Service
NgbitFlashConfig Interface

Indeterminate bar

A material style indeterminate progressbar useful to indicate to the user that something is happening.

Example

The bar by default is indeterminate which is useful to indicate loading states. If 'indeterminate' is set to false, the bar becomes a usual progress bar and the progress value can be set via the 'progress' attribute. The bar integrates seamlessly with the Bootstrap CSS framework and accepts any of bootstrap's contextual classes. This is entirely optional!

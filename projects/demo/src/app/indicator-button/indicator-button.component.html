Indicator button

Buttons that can toggle an indicator signalling to the user that something is happening.

Examples

Click to show indicator

There are 7 different variants of the indicator button you can choose from:

  • Notch
  • Eclipse
  • Bottle
  • Arrow
  • Pulse
  • Disk
  • Circle
Eclipse button Bottle button Arrow button Pulse button Disk button Circle button

API

NgbitIndicatorButtonModule Module
Ngbit[variant]IndicatorButton Component

Replace [variant] with any of the 7: 'Notch', 'Eclipse', 'Bottle', 'Arrow', 'Pulse', 'Disk' or 'Grid'.

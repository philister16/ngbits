Indicators

Small components with spinners and other animated bits to indicate to the user that something is happening. All components rely entirely on inline SVG so you don't have to import any 3rd party icon libraries.

Grid indicator

API

NgbitIndicatorsModule Module
Ngbit[Variant]IndicatorComponent Component

Replace [Variant] with any of the following 10: 'Notch', 'Eclipse', 'Bottle', 'Arrow', 'Pulse', 'Circle', 'Ripple', 'Cube', 'Dots' and 'Grid'.

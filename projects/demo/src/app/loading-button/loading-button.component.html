Loading button

Button components that can change between text and a loading indicator while preserving the initial size.

Examples

Click to load

There are 10 variants of the button corresponding with the 10 different Ngbits indicators. All loading indicators are in pure inlined SVG. You do not need any icon or font libraries.

Notch loading Eclipse loading Bottle loading Arrow loading Pulse loading Circle loading Ripple loading Cube loading Load with grid

API

NgbitLoadingButtonModule Module
Ngbit[variant]LoadingButton Component

Replace [variant] with any of the 10: 'Dots', 'Notch', 'Eclipse', 'Bottle', 'Arrow', 'Pulse', 'Circle', 'Ripple', 'Cube' or 'Grid'.

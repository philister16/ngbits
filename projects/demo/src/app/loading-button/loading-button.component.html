Loading button

Button components that can change between text and a loading indicator while preserving the initial size.

Examples

Click to load

There are a number of variants of the button. All loading indicators are in pure inlined SVG thanks to this amazing project. You do not need any icon or font libraries.

Load with bars Load with rings Load with grid

API

NgbitLoadingButtonModule Module
Ngbit[variant]LoadingButton Component

Replace [variant] with any of the 4: 'Dots', 'Bars', 'Rings' or 'Grid'.

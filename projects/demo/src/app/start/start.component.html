NgBits

Bits and pieces in the form of little libraries for everyday Angular development.

Usage

First install the desired ngbit as a package from NPM.

Each individual ngbit feature consists of a separate module. Import the respective feature's module and add it to the imports array of a module in your application. For example to use the Flash feature do the following:

Optional

The library comes with a number of default stylings which make use of the popular Bootstrap CSS framework. For these to work, you need to also install and import boostrap.
